# Aulas-pucc

O curso visa atender a uma demanda nacional e internacional por profissionais que apresentem competências e habilidades para trabalhar com o desenvolvimento de sistemas e ferramentas que processem grandes volumes de dados. Os dados processados são utilizados para realizar extração de informações que apoiem o processo de tomada de decisões, nas mais diferentes áreas do conhecimento. O objetivo do curso é formar pessoas para trabalhar com o processo de obtenção de conhecimentos a partir de conjuntos de dados, utilizando ferramentas e técnicas estatísticas e computacionais, além de prover uma formação aprofundada em Inteligência artificial.
